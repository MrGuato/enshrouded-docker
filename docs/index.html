Why This Container

Most Enshrouded server setups drift, go stale, or require rebuilds whenever Steam releases an update. This container keeps the image immutable and updates the server binaries at runtime via SteamCMD.

  • Runtime steamcmd +app_update 2278520 validate
  • No broken latest tags
  • Persistent saves, logs, and config via volumes
DevOps Signals
  • CI/CD builds and publishes images to GHCR
  • Reproducible deployments using Docker Compose
  • Automation-first entrypoint design
  • Runs as a non-root container user
Security Posture
  • No privileged containers
  • Only required UDP ports exposed
  • State isolated to mounted volumes
