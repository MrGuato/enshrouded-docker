Enshrouded Dedicated Server

A production-minded Enshrouded dedicated server built with Docker. Always up to date via runtime SteamCMD updates — no rebuilds, no drift.

Why This Exists

Most Enshrouded server setups rely on manual installs or brittle Docker images that fall behind whenever Steam ships an update.

This project keeps the container immutable and updates the server binaries at runtime using SteamCMD.

  • No broken latest tags
  • No manual updates
  • No image rebuilds for game patches

Quick Start

Pull the image:

docker pull ghcr.io/mrguato/enshrouded-docker:latest
